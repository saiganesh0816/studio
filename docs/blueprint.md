# **App Name**: AttendanceAce

## Core Features:

- Attended Days Input: Allow users to input the number of days they attended.
- Total Days Input: Allow users to input the total number of working days.
- Percentage Calculation: Automatically calculate the attendance percentage based on the inputs.
- Status Display: Display a pass/fail status based on the customizable threshold.
- Threshold Customization: Allow users to set a custom attendance threshold to determine pass/fail status.

## Style Guidelines:

- Primary color: Moderate green (#7CFC00) for a positive, encouraging vibe, suggesting success and achievement.
- Background color: Very light green (#F0FFF0) provides a clean, calm, and unobtrusive background, maintaining focus on content.
- Accent color: Soft lime green (#D0FF14), offering just enough difference to highlight the most important elements.
- Body and headline font: 'PT Sans', a versatile humanist sans-serif suitable for all text.
- Simple, clean icons to represent input fields and status indicators.
- Clean and straightforward layout for easy data input and readability.
- Subtle transitions when calculating and displaying the attendance status.
